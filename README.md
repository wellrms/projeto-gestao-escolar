# Projeto - Gestão Escolar (SQL)

Projeto desenvolvido para demonstrar conhecimentos em modelagem e consultas analíticas utilizando SQL.

## Estrutura

- schema.sql = Criação das tabelas
- data.sql = Inserção de dados
- queries.sql = Consultas analíticas

## Conceitos aplicados

- Modelagem
- Chaves primárias e estrangeiras
- Relacionamento NN
- JOIN
- GROUP BY
- HAVING
- Funções agregadas AVG e COUNT
- Filtros com WHERE
- Ordenação com ORDER BY

## Objetivo

Simular um banco de dados escolar e extrair indicadores como

- Média por aluno
- Média por disciplina
- Ranking de professores
- Identificação de alunos com média abaixo de 6
