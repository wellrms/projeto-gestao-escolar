# Power BI – Dashboard de Livraria

Projeto de análise de e-commerce de uma livraria desenvolvido em Power BI, com foco em KPIs e métricas de desempenho.

## Objetivo

Construir um dashboard interativo para analisar o desempenho de vendas, comportamento de clientes e acessos ao e-commerce.

## Indicadores analisados

- Assinaturas ativas e inativas filtradas por região;
- Média de acessos à plataforma;
- Progresso de vendas por semana;
- Acompanhamento de metas de vendas;
- Perfil dos clientes (idade média, profissão, região e canal de contato preferido).

## Funcionalidades do dashboard

- Navegação entre páginas através de botões interativos;
- Filtros dinâmicos por região e período;
- Visualizações interativas para exploração dos dados.
